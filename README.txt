A plugin to add widget listing categories. Extending Default Wordpress Category Widget

== Description ==

Are you frustrated by the lack of customization options in default Category widget in WordPress? Then this plugin is for you. Works similarly but with several customization options. Cheers !

== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' Plugin Dashboard
1. Select 'ns-category-widget.zip' from your computer
1. Install
1. Activate the plugin on the WordPress Plugin Dashboard
1. Place the 'NS Category Widget' into a Widget area through the 'Appearance -> Widgets' menu in WordPress.

= Using FTP =

1. Extract 'ns-category-widget.zip' to your computer
1. Upload the 'ns-category-widget' directory to your '/wp-content/plugins/' directory
1. Activate the plugin on the WordPress Plugins dashboard
1. Place the 'NS Category Widget' into a Widget area through the 'Appearance -> Widgets' menu in WordPress.
